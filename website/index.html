This is the static version of the site — no chat widget.
NGINX Plus serves this file directly from /var/www/routine-kushikimi/index.html

How this page is served

1. Browser requests https://routine.kushikimi.xyz
2. NGINX Plus handles SSL termination, checks basic auth, serves this static HTML file — no backend involved
3. No API calls This version has zero JavaScript. Everything is plain HTML rendered by the browser.
4. Production version index-with-chat.html adds a chat widget that calls POST /api/chat — proxied by NGINX to Node.js on port 3001, which forwards to Azure OpenAI

Stack at a glance

Web server NGINX Plus on Ubuntu 22.04 AWS EC2
SSL Let's Encrypt via certbot
Auth NGINX basic auth — /etc/nginx/.htpasswd
Static files /var/www/routine-kushikimi/
Chat backend Node.js Express on 127.0.0.1:3001
AI endpoint Azure OpenAI gpt-4o-mini
Chat route POST /api/chat → proxy_pass 127.0.0.1:3001
